# FacePlay - 스노우 스타일 얼굴 인식 앱

브라우저에서 실행되는 스노우/인스타그램 스타일의 AR 카메라 앱입니다.

## 주요 기능

### AR 필터 (20종)
- 동물 필터: 강아지, 고양이, 토끼
- 액세서리: 왕관, 선글라스, 하트, 반짝이
- 캐릭터: 악마, 천사, 광대, 외계인
- 고급 효과: 뷰티, 네온, 픽셀, 홀로그램, 불꽃, 얼음, 무지개, 좀비, 사이버펑크

### 미니게임 (5종)
- 과일 받기: 입으로 과일 받기
- 버블 터뜨리기: 코로 버블 터뜨리기
- 얼굴 레이싱: 얼굴로 장애물 피하기
- 표정 맞추기: 지시된 표정 따라하기
- 헤딩 축구: 머리로 공 튕기기

### 음성 효과
- 실시간 피치 조절 (다람쥐/거인 목소리)
- 사운드 비주얼라이저
- Web Audio API 활용

### 기타 기능
- 사진 캡처 및 저장
- 네이티브 공유
- 갤러리 (로컬 저장)
- PWA 지원 (오프라인, 앱 설치)

## 기술 스택

- **얼굴 인식**: MediaPipe Face Mesh (468 랜드마크)
- **오디오**: Web Audio API
- **PWA**: Service Worker, Web App Manifest
- **그래픽**: Canvas 2D API
